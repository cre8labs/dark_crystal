## Dark Crystal

It's a board game.

This version (branch) is using AngularJS as a JavaScript framework.
Server is based on NodeJS.


( this is still under heavy development )


## Technology stack

* JavaScript
* CoffeeScript
* SCSS
* AngularJS Framework
* NodeJS
* Ruby (RVM) - used by Compass
* Yeoman
* Grunt
* Bower
* Twitter Bootstrap
